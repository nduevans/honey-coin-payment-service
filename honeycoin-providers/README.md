# Payments Take-Home

This repo contains a take-home interview task. Read [TASK.md](./TASK.md) for the full brief.

## Getting started

The `/stubs` directory contains pre-built mock payment provider servers you'll need running alongside your service:

```bash
cd stubs && npm install
```

See [stubs/README.md](./stubs/README.md) for how to run them.
